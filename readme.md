# Vulnerable Branch

This branch is intentionally vulnerable for educational purposes.

## Included vulnerabilities
- SQL Injection (demo login)
- Stored XSS (comments)
- Broken Access Control (admin panel)
- Hardcoded admin role logic
- Missing CSRF protection
- Weak secret key

  ## Branches

- vulnerable: Contains intentionally vulnerable code for demonstration
- secure: Includes fixed and secure implementations
- master: Base version of the project
